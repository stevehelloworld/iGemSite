# CadArmor Design System (Master)

Source of truth for VIS iGEM 2026 wiki UI. Aligns **Impeccable** + **UI/UX Pro Max** guidance with the committed GEMS-inspired student brand.

## Style
Claymorphism · educational · soft 3D · playful science wiki

## Colors
See `DESIGN.md` root tokens. Primary rose `#C94C67`, mint secondary, orange CTA accents.

## Type
Fredoka + Nunito. English only.

## Components
- **Navbar:** sticky, soft cream/rose, dropdown stickers
- **PageHero:** pastel wash + chapter badge
- **PageCover:** full-width colored band with mascot doodle
- **Sticker cards:** thick border + offset shadow
- **Callouts:** full border + tinted fill (no left-stripe-only)
- **Buttons:** pill, cursor-pointer, 200ms hover lift
- **Mascot:** armored bacterium SVG (`CadMascot`)

## Checklist (every PR)
- [ ] English only
- [ ] cursor-pointer on clickables
- [ ] Hover 150–300ms ease-out
- [ ] Contrast ≥4.5:1 body
- [ ] Focus ring visible
- [ ] prefers-reduced-motion
- [ ] Responsive 375 / 768 / 1024 / 1440
