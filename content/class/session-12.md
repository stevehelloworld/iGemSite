## 1. 可發布標準（一起念）

1. 結構清楚  
2. 英文可讀  
3. 圖/表有說明  
4. 有限制與下一步  
5. 無假數據  
6. 風格不跑版  

## 2. 圖片完整步驟

1. 選圖、壓縮  
2. 英文檔名  
3. 放到 `public/images/`  
4. md：

```markdown
## Figure

![Describe what the viewer should see](/images/your-file.png)

**Figure 1.** Caption. Say if coursework-only.
```

5. 檢查 alt 是否有用  
6. 窄螢幕看圖是否過大
