## 1. 建造完成 ≠ 可發布

工程上頁面在，但若：

- 圖沒說明  
- 英文讀不懂  
- 語氣像造假  

則**產品失敗**。

---

## 2. 圖片完整流程

1. 壓縮圖  
2. 放入 `public/images/`  
3. md：

```markdown
![What the image shows](/images/your-file.png)

**Figure 1.** Caption in English. State if practice-only.
```

4. 檢查載入與可讀性  

### 競賽觀念

正式 freeze 圖通常需 iGEM 允許托管；本堂先練流程。

## 3. 英文寫作工坊（完整）

使用句型（DBTL / HP / Problem）各寫一段，貼進頁面。  
互改規則：只改清楚度，不替對方發明實驗。

### 可信語氣對照

| 避免 | 改用 |
|------|------|
| perfectly proved | measured / observed / planned |
| 99% without method | report with method on Results |
| everyone did everything | specific attribution |
