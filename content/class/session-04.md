## 今天最重要
**改文字 → 改 `content/` 裡的 Markdown，不是改 React 長文。**

## 動手
1. 打開 `content/pages/description.md`（或你組認領的頁）
2. 改 `subtitle` 或某一段英文
3. 存檔 → 重新整理瀏覽器對應網址

## Front matter 長這樣
```yaml
---
title: Description
eyebrow: Project
subtitle: One English line
tone: rose
---
```

## 鐵律
1. 正文英文  
2. 沒數據不要寫假結果  
3. 套版提示可以先留著
